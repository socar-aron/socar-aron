# About Me

## 프로그래밍/개발

2025년 12월 기준, 16년째 개발을 해왔다. 모바일 게임 3년 차, 모바일 메신저 6년 차, 카셰어링 서비스 8년 차이다. 주로 겪어 본 기술은 피처폰과 Android, iOS 환경이다. 코딩 자체에 관심이 있다기보다는, 내가 예상하는 미래상으로 인류가 이행해 가는 것을 재미있게 지켜보고 있으며, 그 과정에 소소한 기여를 하기를 원한다. 특히 바람직한 체계를 구축하고 그로부터 다수의 행복에 기여할 수 있는 선순환을 만드는 것을 원한다.

## 보유 기술

Kotlin, JAVA, Swift, Objective-C, C++, C#, Android, iOS, Cocos2d-x, Windows Forms, Git, SVN, Jira, Redmine, Slack
